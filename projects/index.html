My projects

Some of the cool stuff I've built / worked on

DataPulse

DataPulse is a digital marketing tool that helps agencies & businesses visualize the performance of their marketing campaigns in real-time. It pulls data from various APIs on-demand and displays dashboards that can be shared with clients. This project was built using Ruby on Rails and a frontend in React.

View homepage

Kaiwa

Kaiwa is a lightweight & highly customizable solution for adding comments to any website. Users can create comment boxes in Kaiwa, then install them using a simple tag. This is a work in progress.

The backend is written in Ruby on Rails, but the kaiwa.js script is pure JavaScript.
